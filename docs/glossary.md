# Глоссарий

| Термин | Определение |
|--------|------------|
| **B2B** | Business-to-Business — модель бизнеса, в которой клиентами являются другие компании |
| **ЛПР** | Лицо, принимающее решения — сотрудник компании, уполномоченный принимать решения о закупках/сотрудничестве |
| **Аутрич** | Outreach — процесс первичного обращения к потенциальным клиентам (холодные письма, звонки) |
| **Лид** | Lead — потенциальный клиент, контакт которого получен для дальнейшей работы |
| **Верификация email** | Процесс проверки существования и доступности email-адреса |
| **Ниша** | Отрасль или сегмент рынка, в котором работает компания |
| **CSV** | Comma-Separated Values — текстовый формат для хранения табличных данных |
| **SPA** | Single Page Application — одностраничное веб-приложение |
| **ORM** | Object-Relational Mapping — технология связывания БД с объектами приложения (Prisma) |
| **CI/CD** | Continuous Integration / Continuous Delivery — непрерывная интеграция и доставка |
| **PR** | Pull Request — запрос на слияние изменений из feature-ветки в основную ветку |
