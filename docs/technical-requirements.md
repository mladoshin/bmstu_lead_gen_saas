# Техническое задание

## 1. Общие сведения

**Название проекта:** Lead Gen SaaS — Сервис автоматического поиска B2B-контактов

**Цель:** Автоматизация поиска и сбора контактов компаний и ЛПР для B2B-продаж.

## 2. Описание продукта

### Основные функции

1. **Поиск компаний** — по выбранной нише (отрасли) и городу система находит релевантные компании
2. **Сбор контактов ЛПР** — автоматический поиск контактных данных лиц, принимающих решения
3. **Верификация email** — проверка валидности и доступности найденных email-адресов
4. **Экспорт данных** — формирование CSV-таблиц с компаниями и контактами

### Пользовательские сценарии

1. Пользователь регистрируется/авторизуется в системе
2. Указывает параметры поиска (ниша, город)
3. Система запускает поиск компаний с помощью ИИ (OpenAI)
4. Система собирает контакты ЛПР найденных компаний
5. Система верифицирует email-адреса
6. Пользователь скачивает результат в формате CSV

## 4. Требования к данным

### Таблица компаний (CSV)
- Название компании
- Отрасль / ниша
- Город
- Сайт
- Телефон

### Таблица контактов ЛПР (CSV)
- ФИО
- Должность
- Email
- Телефон
- Компания
- Статус верификации email

## 5. Нефункциональные требования

- Время поиска — до 60 секунд на запрос
- Поддержка современных браузеров (Chrome, Firefox, Safari)
- Адаптивный интерфейс
